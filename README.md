# Predicting-User-Retention-for-Streaming-Content
- Conducted data exploration, cleaning, and visualization on a streaming platform dataset, handling missing values using XGBoost-based imputation.
- Developed performance metrics (ROI, Revenue Per Viewer) and performed correlation analysis to uncover factors driving user engagement and retention.
- Built and evaluated a Multiple Linear Regression model to predict user retention, providing actionable insights to optimize content strategy.
